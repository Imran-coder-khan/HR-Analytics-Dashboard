# HR-Analytics-Dashboard
This project provides interactive HR insights through 5 dynamic dashboard pages designed for workforce analysis and decision-making.

The dashboard helps analyze:

Employee attrition trends
Salary distribution
Performance analysis
Hiring insights
Workforce demographics
Department-level KPIs
